# HLPS

This is a PyTorch implementation for our paper "Probabilistic Subgoal Representations for Hierarchical Reinforcement learning" (ICML 2024).

## Dependencies
- Python 3.7
- PyTorch 2.0
- OpenAI Gym 0.22
- MuJoCo 2.1.2

Run the codes with ``python train.py``.
